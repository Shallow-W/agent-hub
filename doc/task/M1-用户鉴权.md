# M1 用户鉴权

## 目标

实现简单的用户名密码注册/登录，支持多用户。

## 子任务

### M1-1 注册接口

- `POST /api/auth/register`
- 请求体：`{ "username": "xxx", "password": "xxx" }`
- 密码 bcrypt 加密存储
- 用户名唯一性校验
- 注册成功自动登录，返回 token

### M1-2 登录接口

- `POST /api/auth/login`
- 请求体：`{ "username": "xxx", "password": "xxx" }`
- 验证密码，返回 JWT token
- token 包含 user_id，有效期可配置

### M1-3 登录/注册页面

- 简洁的登录/注册表单
- 登录成功后跳转到 IM 主界面
- token 存储在 localStorage
- 前端请求拦截器自动附带 token

## 验收标准

- [ ] 可注册新用户，密码加密存储
- [ ] 可登录，返回有效 JWT token
- [ ] 无效用户名/密码返回明确错误
- [ ] 前端登录/注册页面可用
- [ ] 后续 API 请求自动附带 token
